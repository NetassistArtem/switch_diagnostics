Подробное описание

Задержка времени после запуска кабель-теста

Задержка по времени после запуска кабель теста для свичей разных производителей. Указанное время подоброну практическим путем, возможно его изменение в меньшую сторону. Если время задержки будет не достаточно порт после проведения кабель-теста не успеет включиться что приведет к получению не правильных данных, (будут полученны данные по порту в состоянии 'off') Определение времени задержки:

Config::set('timeout_cabletest', $timeout_cabletest);

Переменные

$timeout_cabletest
Инициализатор
= array(
'Eltex' => 15,
'Huawei' => 3
)
