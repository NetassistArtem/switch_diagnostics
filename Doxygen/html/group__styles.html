Подробное описание

Стили для подсветки сообщений

Массив содержащий три варианта стиля для подстветки выпадающих сообщений, принимает стандартные значения warning', 'notice', 'information', если гет запросом не приходят названия классов стилей из биллинга. Определение:

Config::set('style_class',$style_array);
